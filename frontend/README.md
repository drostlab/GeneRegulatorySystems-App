# Frontend

Vue 3 + TypeScript SPA for visualising gene regulatory system simulations.

## Prerequisites

- Node.js 18+
- npm

## Install

```sh
cd frontend
npm install
```

## Development

```sh
npm run dev
```

Starts Vite dev server at `http://localhost:5173`. Expects the backend running on `http://localhost:8000`.

## Build

```sh
npm run build
```

Output in `dist/`.
